Variablen

Die Daten eines Programms muss in Speicherplätzen gespeichert werden. Entscheidend für jeden Datentyp ist seine Speichergröße. Zum Speichern eines Wahrheitswertes oder eines zweiwertigen Wertes wird ein boolean verwendet. Dieser Datentyp kann nur true (wahr) und false (falsch) annehmen. Beispielsweise kann eine Lampe oder Schalter nur an oder aus sein. In einer Steuerung sind das binäre Signale. Als Eingabe wird oft eine Checkbox verwendet. Sie kann nur aktiviert oder deaktiviert sein. Oder ein Button, der gedrückt ist oder nicht.

boolean Wahrheitswerte (true/false)
Lampe an

Zum Speicherrn eines Zeichens wird char verwendet. Die Zeichen dieses Textes könnten in char gespeichert sein.

char einzelnes Zeichen

Zum Speichern von ganzen Zahlen verwendet man Integer. Reicht der Platz nicht aus wird long verwendet. Als Beispiel dafür sind die Pixel eines Bildes. Die Breite sind 46 und die Höhe sind 31 Pixel. Es gibt keine halben Pixel oder eine Nachkommazahl.

int -231bis 231 -1
-2 147 483 648 bis 2 147 483 647
long -263bis 263 -1
-9 223 372 036 854 775 808 bis 9 223 372 036 854 775

Fließkommazahlen werden in float oder double gespeichert. Es gibt Zahlen nach dem Komma. Als Beispiel hierfür habe ich den Tachostand eines Fahrzeuges gewählt. Die untere Anzeige wird betrachtet.

int bis 3.40282347E+38
long bis 1.79769313486231570E+308
